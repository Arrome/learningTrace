#安装目录结构解析

linux下目录结构
--------------

| 目录    | 目录内容  |
|-----    |---------|
|/usr/bin |客户端程序和脚本|
|/usr/sbin|mysqld服务器|
|/usr/lib/mysql|库|
|/usr/include/mysql|包含头文件|
|/var/lib/mysql|日志文件，数据库|
|/usr/share/doc/packages|文档|
|/usr/share/mysql|错误消息和字符集文件|
|/usr/share/sql-bench|基准程序|

文件目录
-----------
PID文件：/var/run/mysqld/mysqld.pid
启动日志文件：/var/log/mysqld.log
mysql执行历史： ~/.mysql_history
