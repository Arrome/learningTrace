# 二进制文件包结构

docker
├── docker                   Docker client
├── docker-containerd
├── docker-containerd-ctr
├── docker-containerd-shim
├── dockerd                  Docker daemon
├── docker-init
├── docker-proxy
└── docker-runc


启动 docker daemon
----------------
如果docker daemon未启动，使用 `./dockerd &` 启动
