# 基本变量类型

布尔型bool
------
长度：1字节
取值：true,false(不能用数字0和1代表)

整型 int/uint
----------
长度：根据平台决定可能32/64位

8位整型 int8/uint8
---------
长度：1字节
取值范围：-128~127/0~255

16位、32位、64位整型
----------------

浮点型 float32/float64
------------------
长度：4/8字节
小数位：精确到7/15小数位

复数 complex64/complex128
----------------------
长度：8/16字节
足够保存指针的32位或64位整数型：uintptr
